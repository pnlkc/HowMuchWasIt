# HowMuchWasIt
<br>

## 얼마였더라 앱 (제작중)
- 이전에는 얼마에 물건을 구매했는지 기억이 나지 않을 때 사용할 수 있는 앱
<br>

## 기술 사항
- 전체적인 앱의 UI 는 Compose를 사용해서 구현
- DI(의존성 주입)을 활용하여 앱 제작
- MVVM 패턴을 사용하여 앱 제작
- 로컬 db는  Room 활용
- 앱의 설정 값은 DataStore를 사용해서 구현
<br>
